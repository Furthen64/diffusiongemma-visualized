DiffusionGemma Visualization Software — Implementation Summary

Built an interactive Streamlit application that visualizes all core concepts of
DiffusionGemma (Google's 26B discrete diffusion language model) using simulated
data. No GPU required.

Files created:

  IMPLEMENTATION.md      — Full implementation plan (tech stack, structure, page specs)
  pyproject.toml         — Project config with streamlit, plotly, numpy dependencies
  app.py                 — Landing page with 6 concept cards and sidebar navigation

  pages/
    1_canvas_denoising.py      — Step-by-step canvas denoising: grid heatmap colored by
                                 entropy, token text display, entropy bar chart with budget
                                 line, step navigation (prev/next/slider), auto-play animation
    2_entropy_bound.py         — Entropy-bound acceptance: sorted entropy bars (green=accepted,
                                 red=rejected), cumulative acceptance curve with budget cutoff,
                                 before/after canvas grid comparison
    3_self_conditioning.py     — Self-conditioning flow diagram (logits→softmax→weighted
                                 embeddings→gated MLP→+embeddings), gate value over steps,
                                 distribution comparison between steps, mean entropy reduction
    4_block_sampling.py        — Block sampling loop: Gantt timeline (prefill/denoise/commit),
                                 KV cache growth bar, per-block detail inspection, assembled
                                 text with block boundaries
    5_encoder_decoder.py       — Encoder/decoder mode toggle with attention mask heatmap
                                 (causal vs bidirectional), forward pass walkthrough, KV cache
                                 read/write state visualization
    6_attention_mechanisms.py  — Three views: single mask (causal/bidirectional), batch mixing
                                 (block-diagonal per-sequence masks), sliding window (causal
                                 and bidirectional with query detail)

  utils/
    __init__.py
    diffusion_sim.py     — Core simulation engine: DiffusionSim class with configurable
                           canvas size, entropy bound, temperature, convergence detection,
                           self-conditioning simulation; mask generators (causal, bidirectional,
                           sliding window, per-sequence batch)
    plot_helpers.py      — Reusable Plotly figure builders for heatmaps, bar charts,
                           acceptance curves, Gantt timelines, KV cache bars
    styles.py           — Shared color palette and CSS injected via st.markdown

The simulation engine models diffusion behavior by blending target distributions
with noise and context signals, producing realistic convergence patterns where
confident positions lock in first and propagate context to neighbors.
